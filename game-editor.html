Stats & Scoring

Configure which stats exist and how the game is scored.

Starting Values & Rules
Hard threshold: Player loses immediately if any single stat falls to or below this value. Set to 0 to disable.
Average threshold: Player loses if the average of all stats falls to or below this value. Set to 0 to disable.
Both can be active at once.
Stats Add or remove stats. Each stat needs a key (no spaces), a label, and a color.

Interaction Types

These are the scenario categories players choose from each turn (e.g. Email, Phone Call). Each type needs a unique ID, an emoji icon, and a display label.

The ID is used internally to match scenarios to types — changing an existing ID will break any scenarios already assigned to it. Safe to change labels and icons freely.
Types Shown to players as cards on the "What's on your desk?" screen
